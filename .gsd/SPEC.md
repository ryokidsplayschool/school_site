# SPEC.md — Project Specification

> **Status**: `FINALIZED`

## Vision

Build a high-SEO, visually stunning static website for **Ryo Kids International Play School** — a playschool, daycare, tuition center, and enrichment hub in Thiruvottiyur, Chennai. The site will showcase all programs (ages 1.5–5 playschool/daycare, grades 1–10 tuition, and extracurriculars), feature parent testimonials, an embedded Google Map, and a Contact Us form — all designed to boost enrollments, elevate local reputation, and reduce repetitive inquiry calls.

## Goals

1. **Increase new student enrollments by 30%** within the first year of launch.
2. **Achieve top-3 local SEO ranking** for keywords like "playschool Chennai," "enrichment center Chennai," "daycare Thiruvottiyur."
3. **Reduce repetitive inquiry calls by 20%** by providing comprehensive, accurate program information online.
4. **Reinforce trust and credibility** through authentic parent testimonials, real activity photos, and professional presentation.
5. **Enable easy contact and discovery** via prominent Contact Us form, clickable phone/email, and interactive map.

## Non-Goals (Out of Scope)

- Full-featured online admissions or custom parent portals
- Dynamic CMS — all updates done via static site rebuilds
- On-premise or app-based modules — web presence only
- User accounts or persistent user data storage

## Users

- **Parents/Guardians**: Discovering programs, assessing facilities, comparing extracurriculars, reading testimonials, contacting the school.
- **Staff/Admin**: Supplying content updates (photos, text) to developer for site refreshes.
- **Prospective Students/Teens**: Learning about the 10th grade special coaching program.
